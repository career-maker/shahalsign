# Shaino - Professional Email Signature

A professional HTML email signature template for Sprint Logistic Services LLC.

## Overview

This repository contains a beautifully designed, responsive email signature suitable for corporate communications. Features include:

- **Clean Executive Design** - Professional blue color scheme
- **Responsive Layout** - Works across all email clients
- **Certification Badges** - Display industry certifications and memberships
- **Complete Contact Info** - Phone, fax, mobile, email, and website
- **Company Branding** - Logo and brand accent colors

## Files

- `index.html` - Main email signature template
- `signature_variant_1.html` - Original signature variant
- `Logo-7.svg` - Sprint Logistics logo
- Certification logos (PNG/SVG formats) for various industry memberships

## Usage

### For Gmail Signature

1. Go to **Gmail Settings** → **Signatures**
2. Click **Create new** signature
3. Open `index.html` in a text editor
4. Copy the entire HTML code
5. In Gmail signature editor, click the **</> code editor icon**
6. Paste the HTML
7. Save changes

### For Other Email Clients

Most email clients support HTML signatures. Simply copy the HTML from `index.html` and paste it into your email client's signature settings.
